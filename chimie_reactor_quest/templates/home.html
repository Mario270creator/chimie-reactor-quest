Publicare ușoară

Gândit să fie rulat local cu un click, dar și publicat pe net.

Local nu mai ai nevoie să instalezi Flask sau alte librării separat, fiindcă sunt incluse în proiect. Pentru publicare, ai fișiere pregătite pentru hosting Python standard și pentru deployment cu Gunicorn. 

# local, pe calculator
PORNESTE_PE_CALCULATOR.bat
# sau
python launch_local.py

# telefon + calculator, pe același Wi‑Fi
python launch_network.py

# publicare pe hosting Python
gunicorn --bind 0.0.0.0:$PORT wsgi:app
